To build it, just run:

```bash
$ make
```

You need to have
[`aspell`](http://aspell.net/),
LaTeX,
[`biblint`](https://github.com/Kingsford-Group/biblint),
[`texsc`](https://rubygems.org/gems/texsc),
and
[`texqc`](https://rubygems.org/gems/texqc)
installed.
